# Installation

## State

To install the core State package, run the following command in your terminal:

```bash npm2yarn
npm i uiless.io/state
```

## Frameworks

For each framework, we provide an additional package that you may want to install along `uiless.io/state`.

### React

```bash npm2yarn
npm i uiless.io/state-react
```
